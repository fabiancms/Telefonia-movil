# 📡 Análisis de Tráfico de Red Móvil con Python y Power BI

## 📖 Descripción del Proyecto

Este proyecto simula y analiza el comportamiento de una red móvil utilizando datos sintéticos generados con Python. El objetivo es recrear escenarios reales de telecomunicaciones para aplicar técnicas de análisis de datos, cálculo de indicadores de desempeño (KPIs) y visualización de información mediante Power BI.

La solución permite modelar usuarios móviles, estaciones base, registros de llamadas (CDR), tecnologías de acceso (3G, 4G y 5G), eventos de congestión y fallas de red, generando un entorno realista para el análisis de tráfico móvil.

Todos los datos utilizados son artificiales y fueron creados exclusivamente con fines educativos, analíticos y de demostración.

---

## 🎯 Objetivos

* Generar datos sintéticos de telecomunicaciones utilizando Python.
* Simular tráfico de una red móvil.
* Analizar el desempeño de tecnologías 3G, 4G y 5G.
* Calcular KPIs relevantes del sector telecomunicaciones.
* Realizar análisis exploratorio de datos (EDA).
